# プレゼンテーションファイル（発表用スライド原稿）

## 1. プロジェクト概要
- システム名: EmployeeManagementWeb
- 目的: 社員情報・部門情報をWebで一元管理
- 主な利用者: 管理者（Admin）、一般ユーザー（User）

## 2. 主な機能
- ログイン/ログアウト
- 社員一覧・検索・新規作成・編集・削除（論理削除）
- 部門一覧・新規作成・編集・削除（所属社員チェックあり）
- プロフィール表示・パスワード変更

## 3. データモデル
- Users: 認証情報とロール管理
- Departments: 部門マスタ
- Employees: 社員情報（部門への外部キー、監査項目）

## 4. 権限制御
- 未ログイン時はログイン画面へリダイレクト
- Adminのみ: 社員/部門の作成・編集・削除
- User: 閲覧中心

## 5. 業務フロー（正常系）
- ログイン成功
- 社員登録（Admin）
- 社員一覧で検索・確認

## 6. 品質担保
- 入力バリデーション（必須、メール形式）
- 重複IDチェック（部門）
- テストケースに基づく機能確認

## 7. 今後の改善案
- 単体テスト/統合テストの自動化
- 論理削除データの管理画面追加
- 監査ログの詳細化
- CSVインポート/エクスポート機能
